# Story Stream Frontend


## Структура

```bash
/src
│── /components         # UI-компоненты (кнопки, модалки, карточки и т.д.)
│── /features           # Логические модули (авторизация, профиль, настройки и т.д.)
│── /pages              # Страницы (экраны)
│── /layouts            # Общие макеты (Header, Footer, Sidebar)
│── /hooks              # Кастомные хуки
│── /services           # API-клиенты (axios/fetch), работа с данными
│── /store              # Состояние (Redux/Zustand/Recoil)
│── /utils              # Утилитарные функции (дата, форматирование)
│── /routes             # Конфигурация маршрутов
│── main.tsx            # Точка входа
```
